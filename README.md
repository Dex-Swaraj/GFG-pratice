# Data Structures in C++

This repository contains implementations of fundamental Data Structures and Expression Conversion algorithms in C++ built from scratch for learning, practice, and interview preparation.

## Implementations

### `Implement_stack_using_array.cpp`

Implementation of a Stack using an array. Supports push, pop, top, and empty operations.

### `Stack_using_Linked_List.cpp`

Implementation of a Stack using a Linked List. Demonstrates dynamic memory allocation and LIFO behavior.

### `Stack_using_Queue.cpp`

Implementation of a Stack using Queue(s), achieving LIFO operations using FIFO structures.

### `Queue_Using_Array.cpp`

Implementation of a Queue using an array. Supports enqueue, dequeue, front, and size operations.

### `Queue_using_Linked_List.cpp`

Implementation of a Queue using a Linked List. Demonstrates FIFO operations without fixed-size limitations.

### `Queue_using_Stack.cpp`

Implementation of a Queue using Stack(s), achieving FIFO behavior using LIFO structures.

### `Infix_to_Postfix.cpp`

Converts an Infix expression into a Postfix expression using a Stack. Handles operator precedence, associativity, and parentheses efficiently.

### `Postfix_to_Infix_Conversion.cpp`

Converts a Postfix expression into its equivalent Infix expression using a Stack. Useful for understanding expression parsing and evaluation.

### `Postfix_to_Prefix_Conversion.cpp`

Converts a Postfix expression into a Prefix expression using a Stack. Demonstrates stack-based expression transformation.

### `Prefix_to_Infix_Conversion.cpp`

Converts a Prefix expression into its equivalent Infix expression using a Stack. Processes the expression from right to left and constructs valid infix expressions.

### `Prefix_to_Postfix_Conversion.cpp`

Converts a Prefix expression into a Postfix expression using a Stack. Highlights relationships between different expression notations.
