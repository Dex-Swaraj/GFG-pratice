# Data Structures in C++

A collection of basic Data Structure implementations in C++ for learning and interview preparationon GFG Practice.

## Files

### `Implement_stack_using_array.cpp`

Implements a Stack using an array. Supports push, pop, top, and empty operations with O(1) time complexity.

### `Queue_Using_Array.cpp`

Implements a Queue using an array following FIFO (First In First Out) order with enqueue and dequeue operations.

### `Stack_using_Queue.cpp`

Implements a Stack using Queue(s). The FIFO behavior of queues is manipulated to achieve LIFO stack operations.
